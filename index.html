The Deep Dark

The Deep Dark is a game I'm currently working on as part of my RIT Game Design curriculum, and I am working both as a programmer and the lead audio designer. We expect to launch the game on itch.io by May, but the work I've done so far to record live foley with my Shure SM7b microphone, as well as producing sound with Ableton synthesizers, has been a ton of fun. This project has felt very different from Changeling VR, as not only have I participated in much of the game's design, but I've had to make all of the sounds and music myself.

Changeling VR

Changeling VR is a game in development at RIT, which I had joined to be a Technical Artist for a part-time co-op. While I was a part of the team, I learned how to work with Unreal Engine's sound mastering system, as well as learn how to evolve upon existing sounds, rewrite dialogue, and create my own track to go with the rest of the game's OST.
